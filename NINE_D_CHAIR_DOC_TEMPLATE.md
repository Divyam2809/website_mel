# 9D Chair Page Content Template

Copy the content below into a new Google Doc.
After creating the doc:
1. Click **File** -> **Share** -> **Publish to web**.
2. Click **Publish** and copy the ID from the URL using the normal edit link.
   - Example: `https://docs.google.com/document/d/YOUR_ID_IS_HERE/edit`
   - Copy `YOUR_ID_IS_HERE`.
3. Open `src/scripts/sync-nined-chair.js` and replace `YOUR_GOOGLE_DOC_ID_HERE` with your actual ID.
